      Pad
      Description Etherpad Instance for 57N/ChaosVPN
      People User:Hibby
      Theme Web
      Website https://pad.57north.org.uk
      Status Active

      Etherpad instance for the space.

      Installed on hackr, MySQL database under Hibby's account on there. To migrate, settings.json file and database must be moved.

      Embedding pads in the wiki

      It is possible to embed pads into the wiki. This can be done with the following code in a wiki article: 

      <eplite id="pad-id-goes-here" />

      To set a custom height for the pad, you can add a height attribute: 

      <eplite id="pad-id-goes-here" height="350px" />

      By default, the pad will be 350px high when embedded.
