      Computer:Nelson.hackr.org.uk

      From 57North Hacklab

      Nelson.hackr.org.uk
      FQDN nelson.hackr.org.uk
      Primary IPv4 54.36.74.24
      Primary IPv6
      Location OVH Datacenter, France
      Contact User:Irl
      Projects Projects:HackrDNS
      Processor 1 vCPU
      RAM 256 MB
      HDD 40GB
      Make/Model XenServer HVM Guest
      Operating System Debian 9

      nelson.hackr.org.uk hosts the authoritative DNS for the hackr.org.uk zone and is the mail handler for the hackr.org.uk domain. It is a virtual machine hosted on fredkin.hackr.org.uk. As these services are critical to the operation of the Hackr infrastructure, access to this host is restricted.

      Name

      This host is named after Stew Nelson. Stewart Nelson is an American mathematician and programmer from the Bronx who co-founded Systems Concepts. He created the Midnight Computer Wiring Society because he wanted to add an instruction to the PDP-1 computer, and the lab administrators had forbidden anyone “not qualified” from messing with the computer hardware.
