      Built Aberdeen
      Description An online data centric system to describe our built environment
      People User:Andyg
      Theme Web
      Website https://57north.org.uk/
      Status Active

      An online data centric system to describe our built environment.

      This is a spin off project from an idea for a grant application done for CivTech 2018 Challenge 5. Se "CivTech_3_application_form_challenge5_SSOFB.pdf" for full details.

      Tools

      • Joomla
      • Joomla custom fields component
      • Google Maps / OSM
      • 3rd party data sources

      Tasks

      • Develop basic data framework
      • Develop map driven Joomla template
      • Develop contributor UI/UX
      • Bring in 3rd party data sources

      3rd party data sources, ideas

      • Wikipedia
      • Land Register of Scotland - https://www.ros.gov.uk/our-registers/land-register-of-scotland
      • Canmore: https://canmore.org.uk
      • Historic Environment Scotland: http://portal.historicenvironment.scot
      • HER: https://online.aberdeenshire.gov.uk/smrpub/master/default.aspx?Authority=Aberdeen
      • SIMD: https://simd.scot

      Refs
