      fredkin.hackr.org.uk is a Citrix XenServer host that forms part of the Hackr infrastructure. This host had a previous life with the FQDN hackr.org.uk acting as the sole host for the Hackr infrastructure.

      • 1 Name

      Name

      This server is named after Ed Fredkin. Ed Fredkin (born 1934) is a distinguished career professor at Carnegie Mellon University (CMU), Pennsylvania, and an early pioneer of Digital physics. According to biographer Robert Wright, the character Stephen Falken in the film, WarGames, was modeled after Fredkin.

      Virtual Machines

      The following virtual machines are hosted on fredkin:

      • nelson.hackr.org.uk
      • moore.hackr.org.uk
      • french.hackr.org.uk
      • gosper.hackr.org.uk
      • dompier.hackr.org.uk
      • greenblatt.hackr.org.uk

      Network

      fredkin has both IPv4 and IPv6 connectivity. The following IP addresses are available to fredkin for use:

      • 188.165.234.143/32 - assigned for XenServer management
      • 178.32.58.91/32 - currently not in use
      • 54.36.74.24/29 - assigned for virtual machines
      • 2001:41d0:2:b58f::/64 - assigned for virtual machines (not yet configured)

      Management

      VMs on this host can be managed using the xe command via SSH or using a tool such as OpenXenManager. See Citrix XenServer for more information on managing XenServer hosts.
