      Printing

      From 57North Hacklab

      Nothing on this page is correct and it needs updating.


      There are two 2D printers in the space provided for use by members. While use is free-of-charge, we do encourage donations in order to keep the printers stocked with ink, toner and paper. These donations may also be used to repair and upgrade the devices. A flatbed scanner is also available.

      Colour Inkjet

      The colour inkjet printer is an HP PhotoSmart 5510 series.

      The required cartridges are the HP 364XL. There are black and colour cartridges seperately.

      This printer has not seen enough usage recently and the head could do with a clean. As a result there is some banding present on prints.

      The suggested donation for this printer is 5p per page.

      Monochrome Laser Printer

      The laser printer can only print in one colour, black.

      The required toner drum is the Lexmark E25022G.

      The suggested donation for this printer is 1p per page.

      Printing

      Internet Printing Protocol

      There is a CUPS server running on unicorn that shares access to both printers. If you're using a Linux, BSD or macOS machine that also has CUPS then you should automatically discover the printer and be able to send print jobs to it.

      Secure Shell

      If you have an account on unicorn you can submit print jobs via SSH. You probably don't want to do it this way, but you can. 

      cat document.ps | ssh user@unicorn.57north.org.uk lpr

      Scanning via the Network

      There is a SANE server running on unicorn that shares access to the scanner. If you're using a Linux or BSD machine, you can use xsane to connect to the scanner and retrieve scans.
