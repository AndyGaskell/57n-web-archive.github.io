      Colours

      Below are the colours used by 57North's branding. Hex values are given along with C expressions that show how each colour is derived from 57.

      57N Grey

      Hex: 0x393939

      (57 << 4*8) + (57 << 2*8) + 57

      57N Red

      Hex: 0x390000

      57 << 4*8

      57N Green

      Hex: 0x003900

      57 << 2*8

      57N Blue

      Hex: 0x000039

      57N Pink

      Hex: 0xFF69B4
